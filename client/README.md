# Client - ESP32 TWS 蓝牙耳机修复工具客户端

基于 PySide6 的 GUI 客户端，用于 ESP32 TWS 蓝牙耳机修复工具。

## 功能模块

### 1. 串口连接
- 自动检测可用串口
- 连接/断开 ESP32 设备
- 显示设备连接状态

### 2. 设备扫描
- BLE 设备扫描
- 显示设备列表（名称、MAC地址、信号强度）
- 支持扫描数量配置
- 日志输出

### 3. 修复工具
- 设备复位
- 固件升级
- 清除配对
- 恢复出厂设置
- 校准传感器
- 重启设备

### 4. Bose TWS 辅助
- 设备信息显示（型号、固件版本、电量）
- 同步时间
- 同步设置
- 读取日志
- 发送命令

### 5. 日志
- 实时日志显示
- 时间筛选
- 关键词搜索
- 导出日志

### 6. 协议抓包
- BLE/HID/AVRCP/A2DP 协议抓包
- 数据包列表显示
- 数据详情查看

## 环境要求

- Python 3.8+
- PySide6

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```
